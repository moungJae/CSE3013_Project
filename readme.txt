Openframe Work

실행 전, txt 파일과 mp3 파일 3개를 /bin/data 에 넣어주도록 합니다.

실행 후, 'l' or 'L' 키 입력 후 map.txt 파일을 더블 클릭

'd' or 'D' 키 (누를때마다 1, 2가 반복) 
1. 화면상에 Player, Enemy가 띄워짐    2. Player, Enemy의 위치 재세팅 
'q' or 'Q' 키  => Memory Free 및 게임 종료

< Player 동작에 관련된 키 소개 >
윗 키 => 윗쪽으로 이동		아랫 키 => 아랫쪽으로 이동
왼쪽 키 => 왼쪽으로 이동		오른쪽 키 => 오른쪽으로 이동
스페이스바 키 => 물풍선을 놓음	Shift 키 => 물풍선을 터뜨림

<Game 에 대한 간단한 소개>
Player's life = 5, Enemy's life = 3
Player가 Enemy 4마리를 모두 처치할 경우 게임 승리, 그 외엔 게임 패배
물풍선을 여러개 놓을 수 있습니다. (Score에 의존하므로 블록들을 제거하여 실행합니다)
또한 물풍선을 밀 수 있습니다.

<주의 사항>
'd' or 'D'키를 누르면 화면상에 Player, Enemy가 띄워짐과 동시에 음악이 나옵니다.
이어폰을 꼽고 실행해주시거나 그렇지 못하다면 불륨을 낮추고 실행해주시기 바랍니다.
